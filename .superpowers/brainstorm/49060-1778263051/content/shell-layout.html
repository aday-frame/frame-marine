App Shell Layout

Which navigation structure should the rebuilt app use? All options use the Frame design system: #080808 background, orange accent, Inter font.

A — Icon Sidebar + Topbar (Recommended)

Narrow icon-only sidebar (matches current app). Topbar holds vessel switcher and user menu. Most screen real estate for content. Familiar to current users. Expandable to show labels on hover.

B — Top Navigation Only

All nav in a single topbar. Vessel switcher on the left, module tabs across the middle. More horizontal space lost to labels. Works well on mobile. Gets crowded with 10+ modules.

C — Expanded Sidebar with Labels

Full-width sidebar always showing module names and vessel switcher. Takes more horizontal space but more readable. Good for desktop-first use. Common in tools like Linear and Notion.
