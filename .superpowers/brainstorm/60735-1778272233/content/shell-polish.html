Changes in the "After" design

  • Font — DM Sans (same as original app) replaces Inter. Tighter tracking, better readability at small sizes.
  • Sidebar — Full text labels + collapsible. Section group labels. User name at bottom.
  • Topbar — Workspace switcher with type dot, breadcrumb trail, search bar, notification icon.
  • Cards — 1px top-edge highlight (the subtle white line at the top of each card — a signature detail of the original).
  • Typography scale — 9px uppercase eyebrow labels, 24px light-weight stat values with tabular nums.
